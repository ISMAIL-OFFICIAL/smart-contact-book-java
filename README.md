# Smart Contact Book

## Overview

Smart Contact Book is a Java console-based application that allows users to manage contacts efficiently. The project was built to practice Object-Oriented Programming (OOP), data structures, searching, sorting, and basic file handling concepts.

## Features

- Add new contacts
- View all contacts
- Search contacts by name
- Update contact information
- Delete contacts
- Sort contacts alphabetically
- Linear Search
- Binary Search
- User-friendly console menu

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Searching Algorithms
- Sorting Algorithms
- Exception Handling

## Project Structure

```text
Contact.java
ContactBook.java
Main.java
```

### Contact.java

Represents a single contact and stores:

- Name
- Phone Number
- Email

### ContactBook.java

Handles:

- Adding contacts
- Updating contacts
- Deleting contacts
- Searching contacts
- Sorting contacts

### Main.java

Provides the menu-driven user interface and handles user interaction.

Compile:

```bash
javac *.java
```
